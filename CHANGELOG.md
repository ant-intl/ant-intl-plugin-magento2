# Changelog
All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-07
Initial release of Antom payment extension for Magento 2.

### Added
- Support for payment methods:
    - AlipayCN: redirects buyer to a page to scan the QR code
    - Credit Card: supports one-time card payment
- Modules:
    - Antom_Core: handles interaction with Antom payment gateway
    - Antom_AdminHtml: provides admin functionality for Antom payment integration
    - Antom_Frontend: provides frontend payment UI components and user experience features
